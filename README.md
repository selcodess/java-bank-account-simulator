# Java Bank Account Simulator

A simple Java console application created to practice Object-Oriented Programming and encapsulation.

This project simulates basic bank account operations such as depositing money, withdrawing money, checking balance, and counting successful transactions.

## Features

- Create bank account objects
- Store account owner name and account number
- Deposit money
- Withdraw money
- Prevent invalid deposits
- Prevent withdrawals greater than the current balance
- Track successful transactions
- Print account information

## Technologies Used

- Java
- IntelliJ IDEA
- GitHub

## Concepts Practiced

- Classes and objects
- Constructors
- Private fields
- Getter methods
- Encapsulation
- Methods
- If / else statements
- Validation
- Basic calculations
